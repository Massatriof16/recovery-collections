FUNGSI Spreadtrum Flash

spreadtrum flash (spd_dump) memiliki banyak fungsi dalam melakukan firmware dumper bukan sekedar Unlock bootloader. alat ini melakukan write/tulis partisi, read/baca partisi, bahkan hapus partisi ini merupakan fungsi secara inti sebenarnya.

Maka dari itu Saya membuat Menu batch file yang bernama menu(Indonesia).bat. dengan ini saya memfilter serta mempermudah dalam penggunannya spd_dump, Menu tersebut memiliki fitur : flashing, backup, restore, ganti slot aktif, force reboot,dan reset frp. Gunakanlah sebaik mungkin alat ini dengan hati hati
